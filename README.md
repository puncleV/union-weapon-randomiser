# Weapon Randomizer Union Plugin

Randomize weapons of all NPCs via Marvin command.

## Usage

1. [Build a project](https://worldofplayers.ru/threads/41466/)
2. Activate Marvin
3. Open console
4. Type `weaponsrandomizer RWPNS`
5. Look at how broken game become
6. ???
7. profit!
